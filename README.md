# Tetris

## 概要
ブラウザ上で遊べるテトリスゲームです。  
スタイルは Bootstrap を使用しています。

## 仕様
1. テトリスのミノ整理  
   - 7種類のミノを定義し、ランダムで落下させます。

2. スコア算出機能  
   - 1行消えたら 1点加算。

3. 時間制限  
   - なし。ゲームオーバーになるまで継続。

## UI
- ボタン（画面下に配置）  
  - 開始 / 一時停止  
  - リセット（一時停止した場合のみ表示）  
  - ミノ回転  
  - 左右移動  

## ゲームの流れ
1. 落ちてくるブロックを配列で管理  
2. ランダムに次のブロックを決定  
3. ゲームオーバー判定  
4. NEXT ブロック表示 

## ブランチ運用
- `main`：デプロイ用  
- `develop`：開発用  
- 各機能ごとにブランチを作成（例：`feature/fall-block`）
