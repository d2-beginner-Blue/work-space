# Tetris

ブラウザ上で遊べるシンプルなテトリスゲームです。  
Bootstrap を使ったシンプルなUIで、NEXTブロックやスコア表示にも対応しています。

---

## 🕹 ゲーム概要

- テトリスのブロック（ミノ）がランダムに落下
- 7種類のブロックを使用
- ブロックが横一列に揃うと行が消え、スコアが加算される
- ゲームオーバーになるまでプレイ可能
- NEXTブロックを表示して落下順を確認可能

---

## 🎮 操作方法

### ボタン操作
- **開始 / 一時停止**：ゲームをスタートまたは一時停止  
- **リセット**：ゲームを初期状態に戻す（一時停止中のみ表示）  
- **回転**：ミノを90度回転  
- **左右移動**：ミノを左右に移動  

### キーボード操作
- **↓**：1段落下（ソフトドロップ）  

---

## ⚙️ ゲーム仕様

| 項目 | 内容 |
|------|------|
| ミノ種類 | 7種類（I, O, T, S, Z, J, L） |
| スコア | 1行消すごとに 1点 |
| 落下速度 | 400ms
| 時間制限 | なし（ゲームオーバーまで継続） |
| NEXTブロック | 次に落ちるブロックを表示 |

## 🧩 開発・ブランチ運用

- `main`：デプロイ用
- `develop`：開発用
- 機能追加や修正ごとにブランチ作成例：
  - `feature/fall-block`
  - `feature/score-update`
  - `feature/next-block-display`

### 開発手順
1. developブランチを最新にする
2. `git switch -c feature/xxxx` で新しい機能ブランチを作成
3. 変更を加え `git add .` → `git commit -m "メッセージ"`
4. プルリクエストで main へマージ
